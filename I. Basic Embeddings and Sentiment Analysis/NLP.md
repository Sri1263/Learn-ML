# NLP - Natural Language Processing

Converting Unstructured (Human Understandable) Text to Structured (Machine understandable) Form

## Use Cases
- Machine Translation - Convert one language to another
- Virtual Assistant / ChatBot - take text input to traverse through Tree decisions
- Sentiment Analysis
- N.E.R. - Named Entity Recognition

## Steps to NLP

1. Tokenization - Breaking a string into chunks
2. Inference:

*a. Stemming*
    Removing Prefixes, suffixes etc to get to the STEM
    'Running'|'Runs'|'Ran' -> Run

*b. Lemmatization*
    'Universal'|'University' ??
    Learns meaning of a word through dictionary definition to arrive at ROOT
    'Better' -> 'Good'
3. Parts of Speech tagging
4. N. E. R.
